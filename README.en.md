# 家教预约小程序

#### Description
基于 Spring Boot + Vue 3 + UniApp 开发的家教预约管理系统，提供师资展示、需求发布、在线预约、作业管理、评价反馈、积分支付等完整功能，支持 PC 管理后台和移动端应用。
